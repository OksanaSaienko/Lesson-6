Команди по роботі з терміналом (Git bash):

  1. Якщо вже є репозиторій на віддаленому сервері, тоді:
    • git clone https://github.com/OksanaSaienko/Lesson-6.git - скачується репозиторій на локальний комп'ютер.

  2. Якщо створюємо репозиторій локально(на компютері), порядок дій наступний:
    • git init - в папці проєкту буде ініціалізований репозиторій, тобто з'явиться папка .git з інформацією про репозиторій.
    • git add (назва вашого файлу) - додає ваш файл до майбутнього коміту.
      Якщо кілька файлів, то пишемо через пробіли всі, які хочемо додати.
      Якщо поставити крапку git add . - добавляться всі файли.
    • git commit -m "коментар" - в лапках прописуємо коментар.
    • git remote add origin https://github.com/OksanaSaienko/Lesson-6.git - команда, щоб додати репозитарій на вказаний сайт.
    • git push -u - команда відправляє репозитарій на сайт.

  3. Якщо треба додати зміни, до вже існуючого репозиторію:
    • git add . - додає файли, які треба відправити на репозиторій.
    • git commit -m "коментар" - створюємо коміт з коментарем.
    • git push - відправяємо.

  4. Щоб отримати репозиторій(або зміни) на свій комп'ютер:
    • git pull - команда, що забирає зміни з віддаленого репозиторію і оновлює файли на моєму комп'ютері.
